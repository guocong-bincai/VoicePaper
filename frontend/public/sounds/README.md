# 音效文件说明

本目录包含听写练习页面使用的音效文件。

## 音效列表

### 1. `typing.mp3` - 打字音效
- **用途**：用户输入字母、数字、空格、Backspace时播放
- **特点**：使用 Web Audio API 播放，性能优秀
- **节流**：60ms 间隔，避免过于频繁

### 2. `right.mp3` - 正确音效
- **用途**：用户答题正确时播放
- **特点**：清脆悦耳，给予正向反馈

### 3. `error.mp3` - 错误音效
- **用途**：用户答题错误时播放
- **特点**：柔和提示，避免挫败感

### 4. `error-feeble.mp3` - 轻微错误音效
- **用途**：备用错误提示音
- **特点**：更温和的错误提示

## 技术实现

音效通过 `useTypingSound` Hook 管理：
- **打字音效**：使用 Web Audio API 的 `AudioContext` 和 `AudioBuffer`，支持快速重复播放
- **反馈音效**：使用简单的 `Audio` 对象，适合单次播放

## 音效来源

这些音效文件来自 Earthworm 项目（MIT License），经过优化处理，确保在各种浏览器中都能正常播放。

## 性能优化

1. **预加载**：在组件挂载前就开始加载音频上下文和缓冲区
2. **节流控制**：打字音效限制在 60ms 间隔内只播放一次
3. **资源释放**：播放结束后自动断开连接，释放资源
